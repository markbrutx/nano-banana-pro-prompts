Ready-to-use prompt templates for common tasks. Replace `[placeholders]` with your specifics.

## Outpainting / Image Extension

### Anime Wallpaper Extension
```
Extend this anime image beyond its original borders to a 16:9 widescreen wallpaper.

PRESERVE:
- Character's design, proportions, and exact position
- Original art style, line thickness, and shading technique
- Color palette and lighting direction
- Anime aesthetic (cel-shading, clean lines)

EXTEND:
- Expand background naturally in all directions
- Add environmental details consistent with the scene's mood
- Create depth with foreground and background elements

OUTPUT:
- Aspect ratio: 16:9 (3840x2160)
- Style: High-quality anime wallpaper
- Composition: Character at rule-of-thirds, not centered close-up
```

### Photo Extension to Landscape
```
Extend [Image] to 16:9 landscape format using outpainting.
Keep the main subject intact at [left/center/right].
Generate new content around edges matching:
- Original lighting and shadow direction
- Color temperature and atmosphere
- Photorealistic quality
Seamless integration, high-resolution output.
```

### Portrait to Full Scene
```
Outpaint this portrait to show full body and environment.
Maintain exact facial features and clothing visible in original.
Extend downward to reveal full figure.
Extend sides to show [indoor/outdoor] environment matching the mood.
Keep consistent lighting from [direction].
```

## Character Consistency

### Same Character, New Scene
```
Using the person from [Image1]:
- Keep facial features exactly the same
- Change expression to [happy/serious/surprised]
- New pose: [standing/sitting/action]
- New environment: [description]
- Lighting: [style matching new scene]
Maintain identity perfectly across the transformation.
```

### Character Sheet / Multiple Angles
```
Create a character reference sheet of the person in [Image1].
Show 4 views: front, 3/4 angle, side profile, back.
Maintain exact facial features and proportions across all views.
Neutral pose, consistent lighting.
White/grey background.
Layout: 2x2 grid or horizontal strip.
```

### Character in Different Styles
```
Transform the character from [Image1] into [art style].
Keep recognizable facial features and identity.
Apply [style] aesthetic: [specific style details].
Same pose and expression as original.
```

## Style Transfer & Fusion

### Apply Style from Reference
```
Apply the visual style from [Image2] to the subject in [Image1].
Transfer: color palette, texture, artistic technique.
Keep: subject's identity, pose, composition.
Blend naturally, maintain coherent lighting.
```

### Environment Swap
```
Place the subject from [Image1] into the environment from [Image2].
Match lighting direction and color temperature.
Adjust shadows to be consistent with new environment.
Maintain subject's original pose and features.
Seamless integration, photorealistic result.
```

### Texture/Material Transfer
```
Apply the texture/material from [Image2] to the object in [Image1].
Keep object shape and form.
Realistic material properties (reflections, shadows).
Consistent lighting across the transformation.
```
